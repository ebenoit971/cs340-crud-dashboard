# MongoDB Dashboard / CRUD Application (CS 340)

## About the Project
This project is a Python-based interactive dashboard developed using Dash and MongoDB. It was created for Grazioso Salvare to help filter and visualize rescue dog data.

The application allows users to:
- Filter dogs based on rescue mission types
- View data in an interactive table
- Visualize breed distribution using a pie chart
- Display selected dog locations on a dynamic map

## Technologies Used
- Python
- Dash / JupyterDash
- MongoDB
- Plotly
- Pandas

## Repository Structure
- Original/: Initial version of the project
- Enhanced/: Improved version with added features and optimizations

## Notes
This project was originally developed in a virtual lab environment. Some functionality may require MongoDB setup to run locally.
